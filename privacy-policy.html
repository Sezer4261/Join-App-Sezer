Privacy Policy

1. General Information

The protection of your personal data is important to us. This Privacy Policy explains how personal data is handled within this application.

This project is a non-commercial demonstration and learning project and does not pursue any commercial purpose.

2. Responsible Entity

The responsible entity for data processing within this project is the project team.

As this is a training project, no official data protection officer has been appointed.

3. Collection and Processing of Data

This application does not collect, store, or process personal data on external servers.

All data entered in the application, such as tasks or contacts, is stored exclusively in the user’s local browser storage.

4. Local Storage

The application uses the browser’s localStorage to store application data such as tasks, contact information, and user-related data.

This data remains on the user’s device and can be deleted at any time by clearing the browser data.

5. Cookies

This application does not use cookies.

6. Third-Party Services

No third-party services, analytics tools, tracking tools, or external APIs are used.

7. Data Security

Since all data is stored locally in the browser, users are responsible for protecting their own devices.

8. User Rights

Users have the right to view, delete, or modify their stored data by managing their browser storage.

Users may stop using the application at any time.

9. Changes to This Privacy Policy

This Privacy Policy may be updated if the project is extended or modified.

10. Contact

For questions regarding this Privacy Policy, please contact the project team.
